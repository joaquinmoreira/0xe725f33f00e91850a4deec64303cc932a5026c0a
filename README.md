Proof of Existence
==================

## Prerequisites

- [Node](https://nodejs.org) 
- [Truffle Suite](https://truffleframework.com)

## Getting Started

Install dependencies
```
npm install
truffle install
```

Start ganache
```
ganache-cli
```

Compile contract
```
truffle compile
```

Deploy contract
```
truffle deploy
```

Start dapp
```
npm run start
```

This will create a server in the port `1234`: `http://localhost:1234`

Run tests
```
npm test # 'truffle test' will work too
```
